# Changelog

All notable changes to the TIL Language extension.

## [1.1.0] - 2025-01-28

### Added
- **Run support (F5)** - Run TIL files directly from VS Code
- **Build support (F6)** - Compile to executable
- **Check syntax (F7)** - Syntax checking
- **Run in Terminal (Ctrl+F5)** - Run in integrated terminal
- **Error highlighting** - Inline error diagnostics
- **Right-click menu** - Quick access to run/build/check
- **Run button** - Play button in editor title bar
- **Settings** - Configure compiler path and optimization

## [1.0.0] - 2025-01-27

### Added
- Initial release
- Syntax highlighting for TIL language
- Code snippets for common patterns
- Language configuration (brackets, comments)
- Support for multi-level system (#[level: N])
